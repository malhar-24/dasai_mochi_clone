🤖 DASAI Mochi Clone – Desktop Robot

A fun desktop companion robot built using ESP32 that displays emotions, plays sounds, and includes productivity + entertainment features like a Dino game, timer, and Pomodoro.

This project is inspired by the popular Dasai Mochi concept—small expressive robots with OLED faces that show emotions and react to interactions .


---

✨ Features

🎭 Animated Expressions

Cute robot emotions displayed on OLED

Sound feedback using buzzer


🎮 Dino Game

Play a simple offline game directly on the device


⏱️ Timer Mode

Basic countdown timer for tasks


🍅 Pomodoro Mode

Helps improve focus using Pomodoro technique


👆 Touch Interaction

Control and interact using touch button


🧠 Motion Detection

Uses MPU6050 for motion-based interaction




---

🧰 Hardware Used

ESP32-C6

OLED Display (I2C)

Buzzer

MPU6050 (Gyroscope + Accelerometer)

Touch Button



---

⚙️ How It Works

ESP32 controls all components and runs the logic

OLED displays animations, UI, and game

Buzzer outputs sound for feedback and expressions

MPU6050 detects movement (for interaction/behavior)

Touch button is used for navigation and input


Similar projects use ESP32 with OLED and buzzer to create expressive robotic companions with animations and sound .


🎮 Usage

Power ON the device

Use touch button to:

Switch modes

Play game

Start timer / Pomodoro


Watch expressions + hear sound feedback
